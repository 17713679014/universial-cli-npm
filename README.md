# 通用脚手架封装

统一规范的脚手架开发，降低新建代码库框架搭建时间，时长从1人日缩短至0.5小时内

### 安装

```shell
# install it globally
$ npm install -g police-cli
# or yarn
$ yarn global add police-cli
```

### 使用

#### Quick Start

```shell
police-cli create <name> [-f|--force]
```


#### Options

* `-f, --force`: Overwrite if the target exists
